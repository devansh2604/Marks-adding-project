## 📌 Features

- ✅ Reads student marks from a `.csv` file
- ✅ Automatically adds up marks for students with repeated entries
- ✅ Outputs results using a simple dictionary
- ✅ Works in Google Colab, Jupyter Notebook, or plain Python
